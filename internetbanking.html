Maskan Internet Banking system Redesign

In this project my day-to-day job included designing icons, colorgraphy, images and also implementing pages with Html, Css for the backend team.



Problem

The current system was in maintenance mode for several years and couldn't meet users' new demands anymore also the banking system had changed and the rules on the system were old so most of the users got disappointed with the system and switched to in-person services again. The client wanted to have a completely new system that supported their users' needs and created a better experience for them.





Typography

Source font was Droid Naskh which is formal and their usual font , I used this typography that was scalable and also pleasant for user.



Color Style

I defined Colors to have consistency in all parts of the project and also would use it for mobile bank too.
I create an excell file with complete description included color of hover button, information texts, modal label and all colrs for different part of project.



UI Elements

I designed UI elements that used frequently and have fixed design to use it in implementation too.
