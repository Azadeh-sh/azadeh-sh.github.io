Maskan Internet Banking system Redesign

In this project, my day-to-day job included user research, prototyping, user interface design, documentation, and implementing web pages. Collaboration with developers, data analysts, and Beneficiaries taught me a lot.

"Because of the security of banking system I don't have some parts of the process"



Problem

The current system was in maintenance mode for several years and couldn't meet users' new demands anymore also the banking system had changed and the rules on the system were old so most of the users got disappointed with the system and switched to in-person services again. The client wanted to have a completely new system that supported their users' needs and created a better experience for them.





Research

Our reserch process was:

  • Gather current data from databases and analyze them
  • Review internal and international internet banking systems and list their features to compare
  • Create a questionnaire and asked some users to help us ,interviewed with users and the Support team
  • Brainstorm internal meetings and with the client team
  • Card sorting to select features that we should have on the new system


Results based on Diagnosis

From this study, our first step was to build new user functionality to be visible on the dashboard in the way that users did their actions easily, So we made wireframes and investigate that by getting client's feedback then We made a prototype and captured users group behaviour.

Iteration

We realized that our users used internet banking differently and they had different requirements so the dashboard got customized based on each user's frequent action by using AI.



after that, we released the new view for 10 percent of users and Analyze the result by database queries and also serverside A/B test. The result was acceptable.

the next step was changing the submenu and other actions such as Login, transferring money, and paying a loan, we had the same process for them too.




The final project Received the Reward for UI/UX design between other banking systems in Iran National Festival of Banks and Financial Institutions.
