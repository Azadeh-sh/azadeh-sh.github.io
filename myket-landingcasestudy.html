Myket Landing page Redesign

Redesigning how users deal with the download button to increase app installation

This was an android AppStore website homepage and when users wanted to have apps first of all they should download the AppStore and install it from the website



The Problem

On the current website first we need to increase the rate of clicking, then there was difference between clicking and installing

Analyzing Current Data and Talk to Users

To understand the problem, I investigate Data and statistics from google analytics, Then I add Microsoft clarity to find out the user's behavior, also I did A/B test by google optimize for the color of the button to get more clicks too. I talked to some users who had Clicked on the button but didn't install it too.

I listed some of their problems:

  • Users didn't realize that the file was downloading
  • Users didn't realize that the file has downloaded
  • The banner image shows different apps and they clicked on them instead of the download button
  • download button color didn't have enough harmony to be seen


Ideation and solution

I noticed that users should be aware of the download situation so I prepared a prototype by changing the text of the button and asked some users to use it and I captured their action then I realized they need something more so I added a notification and we released it for 50% of users, The result was acceptable

after that, we changed the banner because we saw that users needed a less distracted banner and also we changed the button color to attract their attention.
